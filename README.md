# NAILS API 
## Description

The project for a business that provides services. 
Api allows adding access to the companies, make record, saves client
info.

**Technologies stack:**

[NestJS](https://nestjs.com/) for backend

[PostgreSQL](https://www.postgresql.org/) as database

---

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
